## 🌟 Overview

**Smart Campus Forum** is a unified digital platform designed to enhance campus life by integrating multiple services into a single intelligent system.

It eliminates fragmented tools and provides a **centralized solution** for communication, safety, and campus engagement.

> 💡 One platform. All campus services.

---

## 🚀 Core Features

### 🤖 AI Chatbot
- Instant, intelligent responses  
- Context-aware answers using RAG  
- Seamless navigation support  

---

### 📅 Event Management
- Browse campus events  
- Access event details easily  
- Improve student participation  

---

### 🛡️ Safety & Reporting
- Confidential harassment reporting  
- Secure admin access  
- Promotes a safer campus  

---

### 🔍 Lost & Found
- Report lost/found items  
- Track recovery status  
- Organized listings  

---

### 👥 Community Interaction
- Create posts and discussions  
- Share updates  
- Build campus engagement
